## Structure

The overall folder structure is the following:

```
src/
├── app/
│   ├── sub-route/
│   │   ├── components/
│   │   └── page.tsx
│   ├── layout.tsx
│   ├── not-found.tsx
│   └── page.tsx
├── assets/
├── components/
│   ├── sub-group/
│   │   ├── Component1.tsx
│   │   └── Component2.tsx
│   └── Component3.tsx
├── types/
│   ├── type.ts
│   └── index.ts
└── hooks/
    ├── useHook.ts
    └── index.ts
```
